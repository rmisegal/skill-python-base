## פרק 7: RAG - הזרקת ידע ארגוני לבינה המלאכותית

### מטרות הלמידה
- הבנת Retrieval-Augmented Generation לעומק
- תכנון מערכת RAG לארגון
- שיפור דיוק תשובות באמצעות ידע ארגוני

### תכנים עיקריים
1. **מהו RAG?** - שילוב אחזור מידע עם יצירת תוכן
2. **הארכיטקטורה**: Embedding → Store → Retrieve → Generate
3. **Chunking Strategies**: איך לחלק מסמכים
4. **Embedding Models**: השוואה ובחירה
5. **Vector Databases**: Pinecone, Chroma, Weaviate בפועל
6. **Evaluation**: מדידת איכות RAG

### נוסחאות מנהליות
- **Recall**: `Recall = מסמכים רלוונטיים שאוחזרו / סה"כ מסמכים רלוונטיים`
- **Precision**: `Precision = מסמכים רלוונטיים שאוחזרו / סה"כ מסמכים שאוחזרו`
- **F1 Score**: `F1 = 2 × (Precision × Recall) / (Precision + Recall)`

### גרפים ותרשימים
- תרשים זרימה: Pipeline מלא של RAG
- גרף השוואתי: ביצועי Embedding Models
- תרשים בלוקים: ארכיטקטורת RAG ארגונית

### דוגמאות מעשיות

- RAG על תיעוד טכני של מוצרים
- RAG על היסטוריית תמיכת לקוחות

## פרק 9: הפריסה - מעבדה לייצור בעולם האמיתי

### מטרות הלמידה
- הבנת אפשרויות פריסה: On-Prem, Hybrid, Cloud
- קבלת החלטות פריסה מושכלות
- ניהול Containers ותשתיות

### תכנים עיקריים
1. **On-Premises**: יתרונות, חסרונות, דרישות
2. **Cloud-Based**: AWS, Azure, GCP - שירותי AI
3. **Hybrid**: שילוב הטוב משני העולמות
4. **Containers**: Docker לפריסת כלי AI
5. **Environment Management**: .env וניהול קונפיגורציה
6. **Scaling**: איך להתמודד עם גדילה

### נוסחאות מנהליות
- **TCO Cloud**: `TCO = (Compute × שעות) + Storage + Egress + Support`
- **TCO On-Prem**: `TCO = חומרה / 5 + חשמל + קירור + כ"א`
- **Break-Even**: `נקודת איזון = TCO On-Prem / TCO Cloud`

### גרפים ותרשימים
- מטריצת החלטה: Cloud vs On-Prem לפי פרמטרים
- גרף עלויות: השוואה ל-3 שנים
- תרשים ארכיטקטורה: פריסה Hybrid
