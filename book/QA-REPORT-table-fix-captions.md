## Correct Caption Patterns

### Pattern 1: Pure Hebrew Caption
```latex
\caption{תיאור הטבלה בעברית}
```

### Pattern 2: Hebrew Caption with English Terms
```latex
\caption{השוואת מודלי \textenglish{LLM} מובילים}
```

### Pattern 3: Hebrew Caption with `\he{}` Wrapper (in English context)
```latex
\caption{\he{מחירי מודלים נפוצים}}
```

### Pattern 4: Caption with `\en{}` for Short English
```latex
\caption{השוואה בין \en{MCP} ל-\en{REST API}}
```

**Key Rules:**
- Every table should have a `\caption{}` (except `longtable` glossaries)
- English words in Hebrew captions should use `\textenglish{}` or `\en{}`
- Caption can appear before or after the table content (both valid in RTL)
- Captions should be descriptive and reference-friendly
